Criando uma página Web utilizando as Tag´s aprendidas

Esta página foi criada com o intuito de colocar em prática o conteúdo aprendido, na primeira etapa do curso. Ela contém um resumo suscinto do assunto tags do HTML.

Antes de aprender sobre o assunto principal dessa página, que são as tags, aprendi sobre a estrutura básica de uma página HTML.

No entanto, como a ideia é descrever e aplicar o conteúdo aprendido na primeira parte do curso, ao mesmo tempo que falo sobre uma TAG, também mostro a aplicabilidade dela.

Um exemplo disso é que, até este ponto da página, já utilizei as tags html, head, meta, title, body, h1, h2, hr, strong, p e i; seguindo o índice abaixo você pode entender para que serve cada uma delas, e também encontrar outras, bastante utilizadas para a construção de uma página HTML.

Uma coisa importante de saber é que muitas tags carregam atributos, que são propriedades para melhorar o funcionamento de cada tag. Existem atributos genéricos e específicos, mas não irei falar sobre eles.

html

Voltar

Essa tag é a que define a extensão do documento, ou seja, define que é um arquivo html.

head

Voltar

Essa tag contém uma parte "não visível" para o usuário, pois ela pré carrega o que será apresentado, e é dentro dela que temos as tags meta e title.

meta

Voltar

São tags especiais, não visíveis para o usuário, que permitem ajudar a página, por exemplo, no ranqueamento do Google.

title

Voltar

Como já se pode imaginar, essa tag traz o título da página.

body

Voltar

Essa tag demarca tudo o que o usuário poderá ver e interagir dentro da página.

h1 até h6

Voltar

Essa família de tags vai desde o h1 até o h6 e serve para dar uma hierarquia aos títulos dentro da página; ou seja, podemos ter um título principal, utilizando a tag h1 e 5 níveis de subtítulos, usando da h2 até a h6. Exemplos de uso são os próprios títulos já utilizados até agora.

hr

Voltar

Essa tag demarca a linha horizontal que você pode ver no início da página.

Strong

Voltar

Essa tag dá a característica de negrito ao texto que ela envolve, mas mais do que isto, essa característica também deixa o texto mais semântico.

p

Voltar

Esta é a tag que utilizamos para demarcar os parágrafos dentro da página.

i

Voltar

Esta tag, quando aplicada, dá ao texto a característica de itálico.

ul

Voltar

Com esta tag podemos criar uma lista não ordenada, ou seja, sem ordem específica, mas apenas com marcadores, como por exemplo, o nosso índice.

ol

Voltar

Já com esta tag, criamos uma lista ordenada, ou seja, podendo numerar dando classes e subclasses, como no exemplo abaixo:

  1. Primeiro item
  2. Segundo item
  3. Terceiro item
    1. Primeiro item dentro do terceiro item

li

Voltar

Esta é a tag que insere cada elemento dentro das listas, ou seja, ao utilizar as tags ul ou ol, apenas definimos qual é o tipo de lista, mas para que haja elemntos dentro dela, inserimos a quantidade necessária de li´s.

blockquote

Voltar

Essa tag de texto serve para fazermos citações, como a do exemplo abaixo.

Lorem ipsum dolor sit amet consectetur adipisicing elit.

u

Voltar

Essa tag foi usada no texto acima para deixá-lo sublinhado.

mark

Voltar

Essa tag dá um destaque no texto.

sup

Voltar

Essa tag é utilizada para colocar um texto de forma superior aos demais, muito usada para esse efeito: Dio R.

sub

Voltar

Essa tag é utilizada para colocar um texto de forma inferior aos demais, muito usada para esse efeito: H2O.

a

Voltar

Essa tag é utilizada para linkar coisas entre si, ou seja, dar a navegabilidade do site, seja entre elementos da própria página, ou entre outras páginas. Por exemplo, se você clicar em Índice você será direcionado para o Índice.

small

Voltar

Essa tag é utilizada para colocar um texto de forma pequena com relação aos demais.

font

Voltar

Essa tag é utilizada para definir as configurações da fonte, como, por exemplo, tamanho, cor, entre outras.

del

Voltar

Essa tag é utilizada para colocar uma linha de forma a representar um texto excluído.

abbr

Voltar

Essa tag é utilizada para colocar uma abreviação em algum texto.
